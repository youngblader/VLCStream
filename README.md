## Getting Started

Follow this instruction to run all the system on your machine.

Clone the repo
```
git clone https://github.com/youngblader/VLCStream
```

After cloning this repo, install pods, open Xcode and build project

## Built with
 
* Swift
* UIKit, SnapKit
* VLCKit
